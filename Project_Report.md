# GamingX Project Documentation
## Modern MERN Stack Redesign & Integration

### 1. Project Overview
This project involved the transformation of a static gaming website into a modern, full-stack **MERN (MongoDB, Express, React, Node.js)** application. The primary goal was to implement a premium "Light Mode" aesthetic with advanced interactive UI features and a cloud-based database.

### 2. Technology Stack
- **Frontend:** React.js, Vite, Axios, Modern CSS3
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas (Cloud)
- **Deployment Ready:** Vercel (Frontend), Render (Backend)

### 3. Key UI Features
- **Neural Assistant v4.0:** An advanced AI-themed chatbot UI for user interaction.
- **3D Tilt Product Cards:** Interactive hover effects using CSS perspective and transforms.
- **Custom Glowing Cursor:** A hardware-accelerated glowing ring that replaces the standard mouse cursor.
- **Scroll Progress Bar:** Real-time visual feedback of page scroll depth.
- **Digital Rain & Floating Orb:** High-performance background animations using CSS keyframes.

### 4. System Architecture
1. **User Interface:** React components fetch data dynamically from the API.
2. **API Layer:** Express server handles requests and communicates with the database.
3. **Data Layer:** MongoDB Atlas stores product information and newsletter subscribers securely in the cloud.

### 5. Setup & Installation
#### Backend
1. Navigate to `/backend`
2. Run `npm install`
3. Create a `.env` file with your `MONGO_URI`
4. Run `node server.js`

#### Frontend
1. Navigate to `/frontend`
2. Run `npm install`
3. Run `npm run dev`
